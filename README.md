# https://lxqjss.github.io  [点击查看效果]( https://lxqjss.github.io/)
### 20170524 初次提交
基于openlayer的便捷地图浏览程序

#### 底图采用如下:
- 谷歌中国提供的最新影像数据
![](/img/normal.png)
- arcgis发布的nasa夜间卫星影像
![](/img/nightlight.png)
- 高德普通地图

#### 实现功能点

1. 基本的地图浏览交互功能
2. 地图量算功能
3. gcj-02坐标展示
4. 底图切换
